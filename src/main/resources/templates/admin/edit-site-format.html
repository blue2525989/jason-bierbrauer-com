Edit various site features.


This section is for editing the title displayed in the browser, the icon displayed, background color, and the meta description and keywords that search engines use to index your web site.

The title is what will be displayed in the browser tab. Also the icon will be displayed there too. Make sure to keep the icon file size small to reduce page load times. Background color is entered starting with a # and followed by 6 numbers and or letters.

The description is limited to around 150 characters so be brief. The keywords need to be separated by commas, be specific and use buzz words.
